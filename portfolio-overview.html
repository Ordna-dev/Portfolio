Sistema administrador de gimnasio (Gimnasio Bonita)

Este proyecto es de autoría mía y de Carlos Medina Sahagún.

Este proyecto consiste en el desarrollo de una aplicación web diseñada para optimizar la administración de Gimnasio Bonita, facilitando tanto la gestión de operaciones internas como la interacción con los clientes. Con una interfaz amigable y eficiente, la aplicación centraliza las funciones administrativas, permitiendo al equipo del gimnasio realizar tareas como el registro y modificación de empleados, clases y horarios, así como la gestión de clientes y sus membresías.

El sistema tiene como fin mejorar la eficiencia en el manejo de los procesos administrativos del gimnasio, tales como la planificación de clases, control de asistencia, registro de pagos y administración de membresías. Además, los usuarios pueden consultar la oferta de clases, inscribirse, y llevar un control de sus asistencias y pagos.

Las tecnologías utilizadas para este proyecto son:

Backend: Laravel y MySQL, para garantizar una estructura robusta y segura en el almacenamiento y procesamiento de datos. Frontend: Laravel Blade, CSS, HTML, JavaScript, JQuery y Ajax, proporcionando una interfaz intuitiva y responsive que facilita la navegación y el acceso rápido a todas las funciones.

Sistema educativo acerca de objetos peligrosos domésticos (KnownHome)

Este proyecto es de autoría mía, de Carlos Medina Sahagún y de Alejandro Díaz Raygoza.

KnownHome es una innovadora aplicación que busca educar a niños sobre los riesgos de los objetos cotidianos en el hogar. Esta plataforma permite a estudiantes y educadores identificar objetos en imágenes y conocer su nivel de riesgo basado en datos reales de accidentes. Mediante el reconocimiento de objetos en fotografías, el sistema muestra información sobre la peligrosidad de estos elementos, fomentando una cultura de prevención y seguridad.

El propósito de KnownHome es utilizar la tecnología para facilitar el aprendizaje en seguridad doméstica de una manera interactiva. Los estudiantes pueden capturar fotos de objetos usando un dispositivo móvil, los cuales son procesados para mostrar información relevante sobre el nivel de peligro de cada objeto. De esta manera, los niños aprenden a identificar y evitar riesgos, con una aplicación que estimula el autoaprendizaje y la responsabilidad en un ambiente controlado.

Las tecnologías utilizadas para este proyecto son:

Backend: Python Flask para la gestión de la API y procesamiento, MySQL como base de datos para almacenar datos de usuarios y objetos, y TensorFlow para el modelo de redes neuronales que realiza el reconocimiento de objetos en las imágenes.

Frontend: Angular + Ionic, SCSS, HTML y TypeScript, creando una interfaz intuitiva y adaptable tanto para web como para dispositivos móviles, permitiendo una interacción ágil entre estudiantes y docentes.

Extras: Orange Data Mining para la clasificación de objetos en grupos de riesgo y Python para el desarrollo de algoritmos de procesamiento y limpieza de datos.
